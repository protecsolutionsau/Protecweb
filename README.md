# Protec Solutions Website

A modern, responsive business website for Protec Solutions, built to showcase services, industry expertise, and contact capabilities.

---

## 🚀 Features
- **Responsive Design** – Optimized for both desktop and mobile devices.
- **Custom Branding** – Integrated Protec Solutions logo, colors, and typography.
- **Services & Industry Focus** – Clear presentation of offerings for Government, Education, and Financial Services.
- **Contact Form** – Allows clients to send inquiries directly to the business email.
- **Custom Domain** – Live at [protecsolutions.com.au](https://protecsolutions.com.au).
- **Integrated Hosting** – Deployed via Netlify with continuous deployment from GitHub.

---

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript, React
- **Hosting:** Netlify
- **Domain:** Custom domain (`protecsolutions.com.au`)
- **Email Service:** Resend (via Supabase Edge Functions)
